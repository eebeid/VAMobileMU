Mobile Course Assistance

This help page will only appear the first time you access the mobile version of this course. If you need to reference the help page later, select the "Help" button at the bottom of the page or from the main menu. When you are finished reviewing this page, select the Menu button at the bottom of this page.

User Interface

The interface houses the user controls and is the display platform for the course content.

  • Main Menu: The Main Menu provides direct links to the sections in this course. You may return to the main menu at any time by selecting the Home button.
  • The Final Exam will be accessible once all sections have been visited.
  • Help Button: The Help button is located on the main menu of the mobile course. Selecting this button will open an additional screen that provides information regarding system requirements, interface features, and course interactivity. Selecting the Back button from this page will return you to the page you were viewing when you selected the Help button.
  • Resources Button: Selecting the Resources button will open an additional screen containing all resources, source documents, and Web addresses for this course.
  • Navigation Buttons: Select the Next button to navigate forward to the next screen, and the Back button to return to the previously viewed screen. Select the Home button to return to the main menu.
  • Exiting the course: Close the browser tab or window to exit the course.

Interactive Features

Interactive features used throughout the course include:

  • Hyperlinks – Selecting a link reveals additional information in any of the following forms:
    • Documents: Links can lead to documents and tables containing testable material.
    • Web Sites: Links can also lead to Web sites outside the course. These Web sites contain useful and/or interesting information relevant to the topic. Content from outside Web sites is not testable material unless it is also found directly in the course content area.
  • Video Player – Video player controls will vary based on your mobile device. To begin playback of the video, select the play button on your video player.
    • Expanding the Video: To expand the video touch the video to display the video player controls, then touch the Expand Arrows button located on the lower right of the controls to expand the screen, touch the Expand Arrows button again to return to the screen to normal view.
